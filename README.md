# 🔌 Calculadora de Consumo de Energia

Este projeto tem como objetivo ajudar usuários a estimar o consumo mensal de energia elétrica de seus aparelhos, além de calcular o custo aproximado em reais.

## 📐 Fórmula Utilizada
\[
ConsumoMensal = \frac{Potência(W) \times HorasDia \times 30}{1000}
\]

Além disso, o custo estimado é calculado multiplicando o consumo mensal pelo valor fixo de **R$ 0,75/kWh**.

---

## 💡 Como Executar
Digite o nome do aparelho: Geladeira
Digite a potência do aparelho em watts (W): 50
Digite o tempo médio de uso diário em horas: 30

--- Resultado ---
Aparelho: Geladeira
Consumo estimado: 45.00 kWh/mês
Custo estimado: R$ 33.75/mês
